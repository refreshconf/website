Ives van Hoorne

Bridging the gap between design prototyping and code

Ives will talk about how design prototyping and production code are increasingly operable. He will speculate on what the future can hold in terms of merging design prototypes and production code and what we already can do with the tools we have today.

Ives van Hoorne

Ives is software developer at Catawiki and student at the University of Twente. He created CodeSandbox, an online editor that focuses on lowering the learning curve of web application development.
