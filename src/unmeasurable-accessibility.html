Unmeasurable Accessibility:
The case for Inclusive Design

When people talk about accessibility they talk about making websites accessible for the use of screen readers (Hi, ARIA!), using good color contrast for visibility and making sure you can navigate the website with the keyboard only. All these things can be tested by the developer either manually or with tools. If coded well the website can even score 100% in accessibility test tools. And yet, be inaccessible for many people.
