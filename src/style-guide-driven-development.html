The Dream of Style Guide Driven Development

Style guide driven development became a reality with components. It enables developers and designers to focus on collaboration and assertive components. With the use of React, CSS-in-JS, Types and Snapshot testing, we were able to almost remove style regressions. In this talk you will get a glimpse on how you can start style guide driven development and how to sell this dream to your project managers.
